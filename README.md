### Hardware Used

- L298 Motor Drivers x2
- HC-SR04 Ultrasonic Sensors x2
- Rasberry Pi 4 (4GB)
- Web Camera (USB-A)
- Wheels + DC3V-6V DC Motors x4
- Wooden Chassis
- 18V of batteries! (12 AA batteries) + Battery Holder
- Portable Charger
- Jumper Cables (MM, FM, FF)
